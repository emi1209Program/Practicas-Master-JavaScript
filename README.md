# Practicas en JavaScript
Este repositorio contiene una colección de ejercicios y prácticas de JavaScript realizados durante el aprendizaje de los fundamentos del lenguaje.
El objetivo del proyecto es practicar conceptos básicos y progresivamente avanzar hacia temas más complejos como DOM, eventos, JSON y LocalStorage.

Fundamentos de JavaScript
* Hola Mundo
* Variables (var, let)
* Constantes (const)

Operadores
 * Condicionales (if, switch)
 * Bucles (for, while)

Funciones
 * Declaración de funciones
 * Parámetros
 * Funciones anónimas
 * Ámbito de variables (scope)
 
Manipulación de datos
 * Strings
 * Arrays
 * Arrays multidimensionales

Ejercicios con estructuras de datos
 * DOM y BOM
 * Selección de elementos
 * Manipulación del DOM
 * Eventos
 * Temporizadores (setTimeout, setInterval)
 * Ventanas del navegador

Almacenamiento y datos
 * JSON
 * LocalStorage
 * Ejercicio práctico de almacenamiento de películas

¿Cómo ejecutar el proyecto?
 No se requiere instalación de dependencias.
 
Requisitos
 * Navegador web moderno (Chrome, Firefox, Edge, etc.)

Pasos
1. Abrir la carpeta del proyecto.
2. Ejecutar cualquiera de los archivos HTML:
 index.html
 o
 pagina.html
